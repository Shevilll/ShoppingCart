## About The Project

Here is a basic Shopping Cart website built with Flask, Sqlite3 and HTML-CSS-JS.

## Getting Started


### Installation

1. Clone the repo

```sh
https://github.com/Shevilll/ShoppingCart.git
```

2. Run:
```sh
pip3 install requirements.txt
```
